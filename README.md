## AutoAlpha X - Your AI Wealth Operator

AutoAlpha X is a live AI-powered wealth operation app that turns users into a one-person, AI-assisted hedge-fund style operator.  
It is designed for WaveHack with a strong product-first flow:

- Identity-based onboarding
- Goal-based finance setup
- AI strategy generation
- Live market intelligence
- Explainable signals
- Smart risk shield
- Multi-mode execution (manual, assist, autopilot)

No Docker is required. The app runs directly as a local web app.

## What the app does

AutoAlpha X helps users go from confusion to action without trading jargon:

1. Understand investor psychology first (Safe Builder / Smart Opportunist / Aggressive Alpha Hunter).
2. Attach a concrete money goal (daily gains, capital growth, long-term).
3. Build a tailored AI strategy.
4. Read live market conditions in simple language (Market Mood, Liquidity Flow, Risk Climate).
5. Browse an opportunity feed with confidence + risk context.
6. Open explainable AI reasoning for each signal.
7. Choose execution mode and simulation vs live readiness.

## Core features implemented

- **Hero + animated brand section** with modern gradient motion style.
- **Onboarding flow** with investor identity and goal intent.
- **AI strategy creator** from user profile + live market state.
- **Live intelligence dashboard** powered by real market data.
- **Opportunity feed** generated from current market movers.
- **Explainable AI**:
  - Uses OpenAI when `OPENAI_API_KEY` is available.
  - Falls back to rule-based explanation when no key is provided.
- **Smart risk shield message** driven by current volatility/risk climate.
- **Execution mode control**:
  - Manual
  - AI Assist
  - Full Autopilot (UI mode logic)
- **Simulation mode switch** for beginner safety.
- **Daily AI report summary** generated from current market context.

## Tech stack

- Next.js 14 (App Router)
- React 18
- API routes inside Next.js
- Live data source: CoinGecko public APIs
- Optional model inference: OpenAI Chat Completions API
- Optional future integration: SoSoValue API and SoDEX docs endpoint

## Project structure

- `app/page.js` - Main product experience and UI flow
- `app/globals.css` - Theme, animation, and UI styling
- `app/api/market/route.js` - Live market snapshot endpoint
- `app/api/strategy/route.js` - AI strategy generation endpoint
- `app/api/explain/route.js` - Explainability endpoint (OpenAI + fallback)
- `app/api/report/route.js` - Daily report endpoint
- `lib/market.js` - Market intelligence logic and strategy builder
- `.env.example` - Environment configuration template

## How it works

1. Client loads `/api/market` for live market state.
2. User chooses identity + goal.
3. Client calls `/api/strategy` to build a personalized strategy.
4. Opportunity feed is rendered from live movers and confidence scoring logic.
5. On "Why this signal?", client calls `/api/explain`.
6. If OpenAI key exists, explanation is generated by model; otherwise a deterministic rules engine explains it.
7. `/api/report` returns a daily summary from current signal + risk context.

## Local setup (without Docker)

### 1) Install dependencies

```bash
npm install
```

### 2) Configure env

Copy `.env.example` to `.env.local` and fill values:

```bash
OPENAI_API_KEY=your_openai_key
SOSOVALUE_API_KEY=your_sosovalue_key_optional
SOSOVALUE_BASE_URL=https://api.sosovalue.com
```

### 3) Run app

```bash
npm run dev
```

Open `http://localhost:3000`.
